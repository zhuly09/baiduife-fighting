## 任务描述

到3月14日正式任务开放之前，完成一个团队页面的设计与开发，目标为：

- 团队一起尝试一个简单的页面是如何制作的
- 磨合团队熟悉度，了解彼此，清楚队员各自的技术能力
- 熟悉各项工具的使用
- 熟悉Git等版本工具的使用

- 每个队员都参与其中
